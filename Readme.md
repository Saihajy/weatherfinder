Application: Weather Finder

Features: 
- Can determine the weather in the top 10 cities in Cananda 
- Can determinte the weather in few other major cities within Cananda

How to run it:
(1) To run locally you need to first download the repo locally, and then run the WeaetherfinderApplication.java file
(2) Then go to http://localhost:8080 to view the site locally
